# Nishikigi
> 一个基于 Onebot 11 协议实现的 QQ 校园墙自动运营系统

> [!NOTE]  
> 推荐搭配 [Napcat](https://github.com/NapNeko/NapCatQQ) 使用

# 使用
1. 修改 `src/config.py`
2. 新建虚拟环境
3. 安装依赖 `$ ./install.sh`
4. `$ ./run.sh &`
